# Sistema de Matching de Talentos com IA Generativa

Sistema desenvolvido como Trabalho de Conclusão de Curso do MBA em 
Engenharia de Software — USP ESALQ.

## Descrição

Dois módulos sequenciais para apoio à alocação de talentos em consultorias:
- **Módulo 1:** extração automática de papéis a partir de propostas comerciais
- **Módulo 2:** avaliação de aderência semântica de candidatos por papel

## Requisitos

```bash
pip install anthropic pymupdf pandas
```

## Execução

O sistema foi desenvolvido para rodar no Google Colab.
Abra o arquivo `matching.py` no Colab e execute a função `executar_matching()`.
